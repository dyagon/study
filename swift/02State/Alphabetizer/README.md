# Alphabetizer

拖动单词牌，把它们按**英文字母顺序**从左到右排好，然后点提交。

## 如何运行

在 Xcode 中打开 `Alphabetizer.xcodeproj`，选择 **My Mac** 或 **iOS 模拟器**，使用 **⌘R** 运行。

## 基本规则

1. **每局 3 张牌**：每张牌有一个英文单词（部分难度下只显示图标，见下文）。
2. **顺序判定**：以每张牌**中心在画面上的左右位置**为准——**最左**的牌视为第一个，**最右**的视为最后一个。系统按从左到右比较单词的**字典序**（`lexicographic`）判断是否已排好。
3. **提交**：只有在提示为「Place the tiles…」时可以提交；提交后会短暂显示对错反馈，约 2 秒后自动进入下一局。
4. **每局换新词**：无论上一局对错，下一局都会从**当前所选词汇包**里重新随机抽出 3 个词（且初始顺序保证**不是**已经排好的顺序）。

## 得分与记录

- **Score**：每次**排对**并提交，+1 分；在 **Strict / Strict + icons** 下，排错会 **−1** 分，分数不会低于 0。
- **Record**：界面会显示 **胜场数 / 总尝试次数** 以及 **胜率**（百分比；尚未提交过时显示为 `—`）。

## 词汇包与「掌握」

- 游戏内有多个**词汇包**（如 Land & trail、Ocean life、Barnyard），可在顶部菜单切换。
- 对**当前尚未标记为已掌握**的包：在同一包下连续 **3 次**提交且都排对，即视为**完成该词汇包的掌握**；中途若有一次排错，该包下的**连续胜利计数会清零**。
- 掌握一个包后，会出现祝贺提示，并自动切换到**下一个尚未掌握**的包。全部掌握后仍可继续玩（随机包），已掌握的包也可随时选回练习（此时连胜不再用于「掌握」进度）。

## 难度模式

| 模式 | 说明 |
|------|------|
| **Standard** | 排错不扣分；每局新词。 |
| **Strict** | 排错扣 1 分；每局新词。 |
| **Icons only** | 牌面只显示 emoji，不显示单词；计分规则同 Standard。 |
| **Strict + icons** | 只显示 emoji；计分规则同 Strict。 |

界面下方会有一行当前模式的简短说明。在显示输赢反馈的约 2 秒内，难度与词汇包菜单会暂时不可用，避免与自动进入下一局的逻辑冲突。

## 反馈含义

- 排对：整张牌会翻到显示对勾；提示胜利文案。
- 排错：只有位置也正确的牌会显示对勾，其余保持正面；提示再试类文案（随后仍会换新一局）。
